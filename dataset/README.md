# dataset

This is the dataset generator.



## Instructions

1. Add all full scrapes to `raw_scrapes` folder.
2. Run `compiler.py`. This will generate `scrapes.csv`.



## Naming Convention

For all full scrapes, follow this naming convention: `<name>_<type>_<region>.txt`.

Examples:

* `ALECO_public_5.txt`
* `IEC_private_4a.txt`
